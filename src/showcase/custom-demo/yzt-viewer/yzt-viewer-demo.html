API

[viewer]

参数 说明 类型 默认值
[viewer-content] 用于标识图片的内容，当动态添加图片的时候，动态初始化viewer Directive
#content 图片内容标识，如果图标使用ngFor方式循环数组展示，请在父级元素加上#content （ngFor模板初始化比ngOnInit晚）
originalAttr 原图路径属性，非必须 String name
