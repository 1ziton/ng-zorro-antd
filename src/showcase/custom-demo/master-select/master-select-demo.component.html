yzt-master 师傅选择

使用nz-select组件封装师傅选择组件

如何使用 #

输入师傅关键字，显示师傅，如果只输入数字会根据电话查找对应的师傅，否则根据姓名模糊搜索

代码演示

默认不传任何属性。

改变valueType可获取不同字段的值，电话/姓名

传valueField可改变下拉选项显示，电话/姓名

valueType和valueField混合使用

使用[customTemplate]传入template模板，可以实现自定义下拉框样式及定制内容

API

master-select

参数 说明 类型 默认值
ngModel 指定当前选中的条目，支持双向绑定 Array（多选）/String/Object（单选） 无
placeholder 输入框默认提示 String 请选择师傅
OptionMode 设置 Select 的模式,默认单选 'multiple' -
valueType 向上返回师傅电话、姓名、对象，'mobile'|'realName'|'object' String 'object'
valueField 下拉选项显示师傅电话、姓名，'mobile'|'realName' String 'object'
width 输入框宽度，支持百分比和像素 String | Number 100%
customTemplate 支持自定义下拉选项，传入模板 TemplateRef -
