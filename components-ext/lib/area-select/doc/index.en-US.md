yzt-area

## API

### yzt-area

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `[(ngModel)]` | 支持双向绑定 | object/string | - |
| `level` | 选择到第几级地址 | string | 3(默认到区) |
| `placeholder` | 空值时默认提示 | string | '请选择' |
| `width` | 宽度 | string | - |
| `height` | 高度 | string | - |
| `valueType` | 返回value的类型('object'/'fullObject'/'wholeName'/'') | string | - |
| `(onChange)` | 选择时回调 | EventEmitter | - |
