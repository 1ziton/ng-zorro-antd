## API

### yzt-loading

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `fullscreen` | 是否全屏遮罩 | boolean | true |
| `message` | 提示语 | string | `加载中` |
| `state` | 外部控制显藏loading | boolean | - |
