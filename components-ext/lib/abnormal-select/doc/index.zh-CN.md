异常选择组件

## API

### yzt-abnormal

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `[(ngModel)]` | 支持双向绑定 | object/string | - |
| `placeholder` | 空值时默认提示 | string | '请选择' |
| `width` | 宽度 | string | - |
| `height` | 高度 | string | - |
| `valueType` | 返回value的类型('object'/'fullObject'/'wholeName'/'') | string | - |
| `(onChange)` | 选择后回调 | EventEmitter | - |
