## Common Use API

### p-tree

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `value` | 树形结构数据 | array | - |
| `selectionMode` | 显示选择模式 `single`,`multiple`,`checkbox` | string | - |
| `[(selection)]` | 已选择的数据 | array | - |
| `loading` | 加载效果 | boolean | false |
| `loadingIcon` | 加载图标 | string | `fa-circle-o-notch` |
